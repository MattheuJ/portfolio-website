On My Time With Google Code Next

Code Next is a program ran by Google which seeks to make computer science, entrpunership, engineering and personal growth skills more accessible to high school students. In Code Next I was a student engineer who attended weekly classes that taught iOS app development, python coding, and robotics. In the summer after my junior year, I joined the programs Web Development club, Team Edge. There, I spent weekdays learning HTML, CSS, and Python to build fully sunctional websites. This culminated in a presentation at a Google office to my peers and Google employees. We visted multiple Google offices where I was able to not only gain technical proficiency but also develop my conversational skills by talking and networking with tech professionals.
